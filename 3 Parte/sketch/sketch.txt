- Suportar o envio de desafios e aceitação de desafios.

	Criar View do tipo IRC em que é possível trocar mensagens (públicas ou privadas) com os jogadores logados
	Além dos jogadores logados podem também	ser listados os jogos disponíveis
	Nesta janela irá ser feita a criação de um novo jogo ou o join a um disponível
	
	Um jogador cria um jogo privado, ocupando de imediato a posição 0 desse jogo (owner)
	Não vai de imediato para o jogo pois vai querer convidar jogadores, ficando no Lobby
	Terá que escolher pelo menos um jogador para o jogo enviando uma mensagem "especial"
	Cada jogador que aceita o convite é direccionado automaticamente para o jogo ficando a aguardar
	O owner poderá começar o jogo assim que o mesmo tiver pelo menos 2 jogadores a aguardar (ele inclusivé)
	A iniciar o seu jogo privado, o owner é direccionado para o jogo, ficando a faltar apenas "Start Game"
